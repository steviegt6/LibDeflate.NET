Forked from <https://github.com/jzebedee/LibDeflate.NET> for use in
<https://github.com/steviegt6/fnb>.

Hand-written C#/.NET bindings for <https://github.com/ebiggers/libdeflate>.

Features an abstracted `Compressor` and `Decompressor` API with implementations
for DEFLATE (`Deflate`-), zlib (`Zlib`-), and gzip (`Gzip`-) file formats using
`libdeflate`.
