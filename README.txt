Hand-written C#/.NET bindings for <https://github.com/ebiggers/libdeflate>.

Forked from <https://github.com/jzebedee/LibDeflate.NET> for use in
<https://github.com/steviegt6/fnb>; this repository is mostly gutted and
features only the bindings to `libdeflate`, `Tomat.FNB.Common` has the managed
APIs.
